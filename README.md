# AsciiArtGenerator

## 概要
文字を画像に変換するプログラムです。  
アスキーアートの出力を主にするつもりで作ったのでこの名前がついています。

## 特徴
* アスキーアートがずれずに出力できる
* 背景色とフォントの色を指定可能
* テキストファイルの読み込みに対応
